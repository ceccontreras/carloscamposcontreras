# Carlos Campos Portfolio

A personal portfolio website built to showcase my work as a software developer, including featured projects, technical stack, real-world data experience, and interests beyond coding.

The site uses a retro terminal / matrix-inspired design and includes dynamic GitHub repository loading, a contact form, interactive media sections, and a fully custom front-end built with HTML, CSS, and JavaScript.

## Live Overview

This portfolio includes:

- About section
- Technical stack
- Data & impact section
- Featured school-related work
- Additional GitHub projects loaded dynamically
- Beyond Coding section with interactive media collage
- Contact form with Formspree integration

## Features

- Custom retro / terminal-inspired UI
- Matrix-style animated side background
- Intro loading screen with animated progress bar
- Dynamic GitHub API project section
- Manual featured project cards
- Custom repo descriptions and screenshots
- Contact form integration using Formspree
- Interactive collage with lightbox and video support
- Responsive layout for smaller screens

## Built With

- HTML5
- CSS3
- JavaScript
- GitHub API
- Formspree
- Google Fonts
